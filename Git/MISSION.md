# Mission: Git Deep Dive

## Why
Go beyond add/commit/push — understand branching strategies, rebasing, conflict resolution, and Git internals. Essential for collaboration on any team.

## Success looks like
- Confidently handle merge conflicts
- Choose between merge, rebase, and cherry-pick
- Use interactive rebase to clean up commits
- Understand Git's internal data model (objects, refs, trees)
- Implement branching strategies (Git Flow, trunk-based)

## Constraints
- Self-paced, hands-on
- Already comfortable with basic Git (add, commit, push, pull)

## Out of scope
- GitHub-specific features (Actions, Projects) — focus on Git itself
- GUI tools — all command-line
