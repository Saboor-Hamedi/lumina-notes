# Mission: Docker & Containers

## Why
Containerize applications for consistent development, deployment, and scaling. Docker is the foundation of modern DevOps — from local dev environments to production orchestration.

## Success looks like
- Write Dockerfiles for any application
- Use docker-compose for multi-service apps
- Manage images, containers, volumes, and networks
- Debug containerized applications
- Understand orchestration basics (Kubernetes)

## Constraints
- Self-paced, hands-on
- Running on Windows — will use Docker Desktop

## Out of scope
- Kubernetes cluster administration (that's a separate topic)
- Docker Swarm (K8s is the standard)
