# Mission: Python Programming

## Why
Python for automation, scripting, data work, and backend development. The most versatile language for DevOps, data science, and web development.

## Success looks like
- Write clean, idiomatic Python scripts
- Automate file processing, API calls, and system tasks
- Understand data structures, OOP, and modules
- Use virtual environments and package management
- Read and modify existing codebases with confidence

## Constraints
- Self-paced, hands-on
- Some programming experience (familiar with basic concepts)

## Out of scope
- Web frameworks (Django, Flask) — focus on core language first
- Data science libraries (pandas, numpy) — separate track
