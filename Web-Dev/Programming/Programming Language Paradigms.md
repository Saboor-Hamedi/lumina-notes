# Programming Language Paradigms

A programming paradigm is a style or way of thinking about programming. Languages often support multiple paradigms.

## Major Paradigms

### Imperative
Describes **how** to do something step-by-step.
- **Procedural**: C, Pascal, BASIC
- **Object-Oriented**: Java, C++, Python, Ruby

### Declarative
Describes **what** to do, not how.
- **Functional**: Haskell, Lisp, Elixir, Clojure
- **Logic**: Prolog
- **Query**: SQL

### Modern Multi-Paradigm

| Language | Paradigms |
|----------|-----------|
| Python | Imperative, OOP, Functional |
| JavaScript | Prototype-based OOP, Functional, Event-driven |
| Rust | Imperative, Functional, Systems |
| Go | Imperative, Concurrent |
| Kotlin | OOP, Functional, Coroutines |
