# Current Linux Level: Absolute Beginner

Established at session start (2026-06-18). User is new to the Linux command line:
- Can list files (`ls`), change directories (`cd`), copy/move/delete files (`cp`, `mv`, `mkdir`)
- Minimal or no experience with pipes, redirection, permissions, processes
- No shell scripting experience
- Running Windows — needs environment setup (WSL / Docker)

**Implications**: Start from the absolute basics — filesystem structure, command anatomy, manual pages. Build up gradually. Each lesson should include running actual commands, not just reading.
