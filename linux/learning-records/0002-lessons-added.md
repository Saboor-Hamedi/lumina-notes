# Linux Curriculum Expanded — Lessons 2-5 Added

Four new lessons created to build from the shell basics into practical DevOps skills:

- **Lesson 2** (0002-files-and-permissions.md) — cp/mv/rm, grep, chmod, chown, rwx model
- **Lesson 3** (0003-pipes-redirection.md) — pipes, redirection, cut/sort/uniq/wc, log analysis patterns
- **Lesson 4** (0004-processes-and-monitoring.md) — ps, top, free, df, kill, background jobs
- **Lesson 5** (0005-networking-essentials.md) — curl, wget, ping, ss, traceroute, SSH, scp

All lessons follow the same structure: theory → hands-on exercise → self-quiz → next steps. The progression builds toward Server/DevOps readiness.

Next planned: Shell scripting (Lesson 6), Package management (Lesson 7).
