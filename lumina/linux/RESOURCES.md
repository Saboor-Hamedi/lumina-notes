# Linux Command Line Resources

## Knowledge (External)

- [GNU Coreutils Manual](https://www.gnu.org/software/coreutils/manual/coreutils.html) — official docs for ls, cp, mv, rm, etc.
  Use for: authoritative reference on basic command behavior
- [Linux Command Line Basics — Ryan Chadwick](https://ryanstutorials.net/linuxtutorial/) — beginner-friendly tutorial series
  Use for: structured learning path from zero
- [Bash Guide — Greg's Wiki](https://mywiki.wooledge.org/BashGuide) — the best free bash scripting resource
  Use for: shell scripting deep dives
- [ExplainShell.com](https://explainshell.com/) — paste any command to see what each part does
  Use for: decoding complex one-liners
- [Linux Journey](https://linuxjourney.com/) — gamified Linux lessons
  Use for: supplemental practice

## Wisdom (Communities)

- [r/linuxquestions](https://reddit.com/r/linuxquestions) — friendly Q&A for Linux learners
- [r/commandline](https://reddit.com/r/commandline) — CLI tips, tricks, and workflows
- [Stack Overflow (linux tag)](https://stackoverflow.com/questions/tagged/linux) — vast archive of solved problems
- [Unix & Linux Stack Exchange](https://unix.stackexchange.com/) — higher-signal Linux Q&A
